Quick reference

The full API in a few lines.

GET /update

Update DNS record to your current IP. Pass domains and token as query params.

GET /update?ip=1.2.3.4

Set a specific IP address instead of auto-detecting from the request.

GET /lookup?domain=my-server

Query the current IP for any LlamaDNS subdomain.

# Update every 5 minutes
*/5 * * * * curl -s "https://llamadns.org/update\
?domains=my-server\
&token=sk_..."
# That's it. No daemon, no agent.

FAQ

Is LlamaDNS really free?

Yes. LlamaDNS is 100% free for personal use and Open-source under the MIT license.

How fast do changes propagate?

DNS records update globally within 60 seconds thanks to low TTL settings.

Do you support IPv6?

Yes. Full AAAA record support. Update IPv4 and IPv6 addresses independently.

How many subdomains can I have?

Each account gets up to 5 subdomains on the free tier.
